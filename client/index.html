Timestamp

Take a string as a parameter that should be a unix timestamp or natural date (ex. January 1, 2016). Returns an object holding both date formats, which will be null if an invalid date was given.

Example usage:

http://ryanjc-api-project.herokuapp.com/api/ts/December%2015,%202015
http://ryanjc-api-project.herokuapp.com/api/ts/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015" }

Who Am I?

Display the IP address, language, and device details for the machine that makes a request.

Example usage:

http://ryanjc-api-project.herokuapp.com/api/whoami

Example output:

{"ipaddress":"68.111.152.79","language":"en-US","software":"Windows NT 10.0; WOW64"}

URL Shortener

Takes a URL as a parameter and hashes it to get a unique code. Use this code to redirect to the given URL.

Example creation:

http://ryanjc-api-project.herokuapp.com/api/shorten/https://google.com

Example creation output:

{"original_url":"https://google.com","short_url":"http://ryanjc-api-project.herokuapp.com/9999"}

Example usage:

http://ryanjc-api-project.herokuapp.com/9999

Redirects to:

https://google.com

Image Search

Takes a search string and returns image results from Google. Add ?offset=_ to paginate through results. Alternatively use the api with "recent" to see recent searches.

Example search:

http://ryanjc-api-project.herokuapp.com/api/imagesearch/ktm 525?offset=10

Example output:

[{"url":"...","snippet":"...","thumbnail":"...","context":"...","alt":"..."}, ...]

Example recent:

https://api-project-ryanjc.c9users.io/api/imagesearch/recent

Example output:

[{"term":"ktm 525","when":"2016-09-13T18:37:10.102Z"}, ...]
